    1. Basic Setup
    1.1 Set Academic Session

    First, ensure the current academic session is set correctly:

    • Go to Management → Current Session & Term
    • Select the active academic session and term
    • This setting affects all exam operations
    1.2 Configure Exam Types

    Set up different types of examinations:

    • Navigate to Examination → Settings → Exam Types
    • Add exam types (e.g., Midterm, Final, Unit Test)
    • Define characteristics for each exam type
    2. Creating an Exam
    2.1 Create New Exam

    Steps to create a new examination:

    1. Go to Examination → Exams → Create New
    2. Fill in basic details:
      • Exam name
      • Exam type
      • Start and end dates
      • Applicable classes
    3. Save the exam details
    3. Managing Question Papers
    3.1 Upload Question Papers

    Process for managing question papers:

    1. Navigate to Question Papers section
    2. Choose paper type:
      • Manual upload (PDF/DOC)
      • Auto-generation (Excel template)
    3. Set paper details:
      • Subject and class
      • Total marks and passing marks
      • Duration and instructions
