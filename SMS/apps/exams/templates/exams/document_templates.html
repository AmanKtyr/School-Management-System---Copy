    Template Usage Guide
    Admit Card Template

    The admit card template is designed for printing student examination entry permits. It includes:

    • Student information section
    • Examination details
    • Schedule table
    • QR code for verification
    • Signature fields

    Recommended: Print on A4 size paper in portrait orientation.

    Question Paper Template

    The question paper template provides a standardized format for creating examination papers. Features include:

    • Properly formatted header with institution details
    • Exam information section
    • Instructions for students
    • Structured sections for different question types
    • Footer with page numbers

    Recommended: Edit in Microsoft Word or compatible word processor.

    Answer Sheet Template

    The answer sheet template is designed for student responses during examinations. It includes:

    • Student information fields
    • Examination details
    • Structured answer spaces
    • Page numbering
    • Space for examiner marks

    Recommended: Print on A4 size paper in portrait orientation.

    Report Card Template

    The report card template provides a comprehensive format for student performance reports. Features include:

    • Student and class information
    • Subject-wise marks table
    • Grade calculation
    • Performance graphs
    • Teacher and principal signature fields
    • Attendance summary

    Recommended: Print on A4 size paper in portrait orientation.

    Attendance Sheet Template

    The attendance sheet template is designed for tracking student presence during examinations. It includes:

    • Exam and subject information
    • Student list with roll numbers
    • Signature columns
    • Invigilator details
    • Summary section

    Recommended: Print on A4 size paper in landscape orientation.

    Marks Entry Template

    The marks entry template is an Excel spreadsheet for efficient entry and calculation of student marks. Features include:

    • Pre-formatted columns for student details
    • Subject-wise mark entry cells
    • Automatic calculation of totals and percentages
    • Grade assignment based on marks
    • Data validation to prevent errors

    Recommended: Use with Microsoft Excel or compatible spreadsheet software.
